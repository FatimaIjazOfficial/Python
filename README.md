# 🐍 Python Programming Fundamentals

This repository contains structured Jupyter Notebooks covering the core fundamentals of Python programming. It is designed to build a strong understanding of Python concepts through practice-based learning.

---

## 📁 Repository Structure

- **Basics.ipynb** → Python basics (variables, data types, syntax)
- **Operators.ipynb** → Arithmetic, comparison, logical, and assignment operators
- **Loops.ipynb** → for loops, while loops, nested loops
- **Data Structure.ipynb** → lists, tuples, sets, dictionaries
- **Functions.ipynb** → functions, parameters, return values, lambda expressions
- **OOP.ipynb** → object-oriented programming (classes, objects, inheritance, polymorphism, encapsulation)
- **File Handling.ipynb** → reading, writing, and appending files
- **Error Handling.ipynb** → exception handling using try, except, else, finally, and custom exceptions

---

## 🎯 Purpose of This Repository

The goal of this repository is to:

- Understand Python programming from basics to intermediate level  
- Practice core programming concepts step by step  
- Improve problem-solving and logical thinking skills  
- Build a strong foundation for advanced programming topics
